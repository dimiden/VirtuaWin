VirtuaWin v4.1 Help

Overview

VirtuaWin is a virtual window manager for Windows 9x/ME/NT/2000/XP/Vista with up to twenty desktops. It’s quick and easy to use with no fancy graphics, just an icon in the system tray indicating current desktop.
